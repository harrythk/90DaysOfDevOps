# What is the difference between git add and git commit?
git add - this command add file to staging
git commit - add the staging file to git repo

# What does the staging area do? Why doesn't Git just commit directly?
Staging area help choose which changes you wnat to commit. If you commit directly, it would insert all the changes you do without choosing.

# What information does git log show you?
git log shows commit id, author and date / time of commit.

# What is the .git/ folder and what happens if you delete it?
The .git folder gets created when you initialize git. It carries all the details of git i.e. commit history, branches, logs etc.

# What is the difference between a working directory, staging area, and repository?
Working Directory - The actual project files on your system where you edit code.
Staging Area - where you select changes to include in the next commit.
Repository - history stored inside the .git/ folder where commits are saved.
